DrawPS
======
DrawPSはC言語用のライブラリです。

プログラムを実行しながら簡単に2D/3Dのアニメーションをリアルタイムに表示することができます。従来の数値計算ではすべての計算を終えてから、その結果をgnuplotやExcelなどを使ってグラフ化することで物理現象を考察していました。DrawPSでアニメーション化をすることで運動の時間発展などを直感的に理解しやすくなります。
